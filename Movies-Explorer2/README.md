# 🎬 Movies Explorer - Professional Dashboard

This is a comprehensive full-stack application for managing and exploring movies, built with the **MERN Stack** (MongoDB, Express, React, Node.js).

## 🚀 Features & Functionality

* **User Authentication:** Secure login and registration using JWT.
* **Movie Management:** CRUD operations (Create, Read, Update, Delete) for movie records.
* **Dynamic Search:** Real-time filtering and search capabilities.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
* **Database Integration:** Seamless connection with MongoDB Atlas.

## 🛠️ Tech Stack & Dependencies

### Backend
* **Node.js**: Runtime environment.
* **Express**: Web framework for the API.
* **Mongoose**: MongoDB object modeling.
* **Cors & Dotenv**: Environment management and security.

### Frontend
* **React.js**: UI Library.
* **Axios**: For API requests.
* **CSS3**: Custom styling and layouts.
